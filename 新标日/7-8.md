# 第7课 / 第8课

### 第7课 重点语法例句

[[[
1. '李{り}さんは 毎日{まいにち} コーヒー_を_ 飲{の}みます。' (小李每天喝咖啡。)
]]]

[[[
2. '李{り}さんは 図書館{と・しょ・かん}_で_ 勉強{べんきょう}します。' (小李在图书馆学习。)
]]]

[[[
3. 'わたしは 毎朝{まいあさ} パン_か_お粥{かゆ}を 食{た}べます。' (我每天早上吃面包或者喝粥。)
]]]

[[[
4. 'コーラとケーキ_を_ _ください_。' (请给我可乐和蛋糕。)
]]]
